# CEI-Social-Media-Analytics

Ce projet repose sur l'utilisation de la librairie Dash pour le développement d'une application multipage
permettant l'analyse de la web réputation d'une marque sur les réseaux sociaux.

Pour lancer l'application, il suffit d'uploader le projet, d'installer les librairies nécessaires, et de lancer
le fichier Dashboardtest.py
